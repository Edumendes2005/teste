# Base Network Airdrop Tracker & Tools

Este repositório foi criado para documentar, testar e automatizar interações com a rede **Base** (Layer 2 da Coinbase), com foco em acompanhar atividades de ecossistema e potenciais critérios de airdrops.

---

## 🚀 Funcionalidades / Objetivos
* **Smart Contracts de Teste:** Deploy de contratos simples na Base Goerli/Sepolia.
* **Scripts de Interação:** Automação de transações básicas (swaps, mints de NFTs de ecossistema).
* **Análise de Dados:** Registro de dApps interagidos na rede principal.

## 🛠️ Tecnologias Utilizadas
* **Git / GitHub:** Para controle de versão e registro de commits.
* **Foundry / Hardhat:** (Se aplicável) para desenvolvimento de contratos.
* **Node.js / Ethers.js:** Para criação de scripts de interação com a blockchain.
